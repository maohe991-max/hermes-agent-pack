# 使用示例

以下演示加载 `agent-squad` 技能后，如何自动匹配子代理执行任务。

---

## 示例 1：代码审查

```
你：skill_view(name='agent-squad')
→ 加载主控文件，准备就绪

你：审查一下这个项目
→ 自动匹配「代码审查官」
→ 按流程：项目概览 → 逐层审查 → 输出报告

👨‍💻 代码审查官：好的，我先看看项目结构和技术栈...
```

## 示例 2：Bug 排查

```
你：生产环境报 500 了，帮忙查一下
→ 自动匹配「Bug 猎人」

🐛 Bug 猎人：有错误日志吗？先看看堆栈信息...
```

## 示例 3：安全审计

```
你：做个安全审计
→ 自动匹配「安全审计员」

🛡️ 安全审计员：开始扫描攻击面，检查配置安全...
```

## 示例 4：架构评审

```
你：评估一下这个技术方案
→ 自动匹配「架构师」

🏗️ 架构师：我先了解业务目标，然后分析现状，给出方案对比...
```

## 示例 5：强制指定子代理

```
(performance-optimizer) 首页加载太慢了
→ 强制使用「性能优化师」

⚡ 性能优化师：先建立性能基准，然后识别瓶颈...
```

## 示例 6：数据库优化

```
(database-admin) 这个查询跑得太慢了
→ 强制使用「数据库管理员」

🗄️ 数据库管理员：看看执行计划，检查索引使用情况...
```
